## Description

This is a simple NestJS project that uses the following technologies:

    Backend Framework: NodeJS

    Language: TypeScript

    Server Framework: NestJS

    Database: PostgreSQL

    Testing Framework: JestJS

    API Documentation: Swagger

## Prerequisites

To run this project you need to have the following installed on your machine:

    NodeJS

    PostgreSQL

## Configuration

the app is running on port 3000 by default

don't forget to include your google maps api key in the .env file

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development on
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```

## Documentaion

you can reach the api documentation from the following link: [http://localhost:3000/swagger](http://localhost:3000/swagger)
